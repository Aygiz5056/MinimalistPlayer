# MinimalistPlayer

Минималистичный музыкальный плеер для Android с прозрачным дизайном.

## Возможности
- 🎵 Воспроизведение музыки с устройства
- ❤️ Избранное (сохраняется в Room Database)
- 🔁 Режимы повтора: нет / всех / одного
- 🔀 Перемешивание треков
- ⏲️ Таймер сна
- 📱 Виджет на рабочий стол
- 🔔 Уведомления с управлением
- 👆 Свайпы и жесты:
    - Свайп влево по списку - открыть плеер
    - Свайп вправо - открыть меню
    - На экране плеера: свайпы влево/вправо для переключения треков
    - Двойной тап - пауза/плей
    - Вертикальные свайпы справа - регулировка громкости
- 🖼️ Обложки альбомов с размытием на фоне

## Технологии
- Kotlin
- Room Database
- MediaPlayer + Visualizer
- Glide
- ViewBinding
- GestureDetector
